## 攻击非常有效并且很简单

请大家尽快升级 node 和 iojs 到最新版本

* node >= 0.12.6
* iojs >= 2.3.3
* iojs >= 1.8.3
node 0.10.x 不受影响。
如果你还在使用 node 0.11.x，那么请尽快升级到 0.12.6
