可以通过这个地址访问到

```
http://alinode.aliyun.com/dist/new-alinode/alinode.json
```

其中
```json
{
  "version": "v4.7.2",
  "date": "2019-03-11",
  "files": "linux-x64,osx-x64-tar",
  "npm": "6.4.1",
  "v8": "6.8.275.32",
  "uv": "1.23.2",
  "zlib": "1.2.11",
  "openssl": "1.1.0j",
  "modules": "64",
  "node": "v10.15.3",
  "notes": [
  	""
  ]
}
```

version 是 alinode 版本，node 是官方 node 版本
