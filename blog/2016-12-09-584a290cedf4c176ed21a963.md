## Node.js
* [power-assert 强大的断言库](https://github.com/power-assert-js/power-assert)
* [cost-of-modules 最慢模块查询](https://github.com/siddharthkp/cost-of-modules)
* [space-hogs 查找占用空间最大的目录命令行工具](https://github.com/dylang/space-hogs)
* [is-my-json-valid 校验 json 格式是否有问题](https://github.com/mafintosh/is-my-json-valid)
* [tingodb 嵌入式的 node.js 数据库，与 mongodb 一致的 api](https://github.com/sergeyksv/tingodb)
* [webpack-bundle-analyzer webpack 打包分析器插件](https://github.com/th0r/webpack-bundle-analyzer)
* [svg-captcha  svg 验证码库](https://github.com/lemonce/svg-captcha)
* [to-capital-case  转换成首字母大写的库](https://github.com/ianstormtaylor/to-capital-case)
* [collect.js 提供了便捷且零依赖的操作数组与对象的多个工具函数，譬如 combine 函数可以将两个数组合并为对象列表，groupBy 可以根据键来进行分组等等。](https://github.com/ecrmnn/collect.js)
* [flubber 是用于平滑形状变换动画的工具库。在 Web 开发中，如果我们需要在基于 SVG 或者 Canvas 构建的两个图形之间进行变换；如果这两个图形本身毫无关联，那么可能会出现异常的中间状态。而 flubber 则内置了一些推测算法来帮我们平滑这个变换过程。](https://github.com/veltman/flubber)
* [yn 将一些值方便的转换为 boolean 值的库，避免踩坑或者冗长的判断](https://www.npmjs.com/package/yn)
