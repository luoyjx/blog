## Summary
@intuitmachine 提出的 AI 工程能力四层框架：Prompt Engineering → Context Engineering → Fine-tuning → Architecture。递进层次，真正竞争壁垒在第3-4层，多数团队止步于第1-2层。

## Key Concepts
- **AI Engineering Layers** — AI 工程能力的四层递进框架
- **Prompt Engineering** — 第1层：基础 LLM 交互优化
- **Context Engineering** — 第2层：管理输入上下文（当前热点）
- **Fine tuning** — 第3层：调整模型权重适配特定任务
- **AI Architecture** — 第4层：系统级 AI 应用架构设计

## Framework
```
Layer 4: Architecture    ← 竞争壁垒
Layer 3: Fine-tuning     ← 竞争壁垒
Layer 2: Context Eng.    ← 当前热点
Layer 1: Prompt Eng.     ← 入门层
```
