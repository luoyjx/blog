## Summary
Google Labs 实验性产品 **Google Opal**，定位为"极简 Coze + 可视化 n8n + 自然语言生成"。用自然语言描述需求即可自动生成节点式 **Visual Workflow**，无需代码、部署或技术背景。实测30分钟完成穿搭应用（含天气API、**Nano Banana Pro** 生图、交互HTML）。在 Cursor/Claude Code（高学习成本）和纯 Gemini（完全黑盒）之间找到平衡点，代表 **AI Democratization** 方向。适合快速验证想法和 **Rapid Prototyping**，不适合企业级复杂应用。实验阶段免费。

## Key Concepts
- **Google Opal** — Google Labs 的自然语言驱动 AI 工作流生成工具
- **No Code AI** — 无需编程即可创建 AI 应用的范式
- **Visual Workflow** — 节点+连线的可视化工作流，自动由自然语言生成
- **AI Democratization** — 降低技术门槛让普通人参与 AI 创新
- **Nano Banana Pro** — Google Gemini 3.0 Pro，Opal 中可免费使用
- **Rapid Prototyping** — 30分钟从想法到可用原型
