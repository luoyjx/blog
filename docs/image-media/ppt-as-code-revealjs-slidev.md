## Summary
PPT as Code 理念：用 Markdown/HTML 代码替代 PowerPoint。两大主流工具：Reveal.js（HTML/JS 高度可定制）和 Slidev（Vue+Markdown 开发者友好）。Git 可追踪，适合 AI 生成内容。

## Key Concepts
- **PPT as Code** — 代码化演示，Git 友好，可编程
- **Reveal.js** — HTML/CSS/JS 驱动，高度可定制
- **Slidev** — Vue.js + Markdown，开发者优先体验

## Comparison
| 工具 | 技术栈 | 适用场景 |
|------|--------|---------|
| Reveal.js | HTML/JS | 高度定制 |
| Slidev | Vue+MD | 开发者日常 |
