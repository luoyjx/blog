## Summary
Claude Code 图表生成插件，自然语言描述转 draw.io 图表（流程图、AWS架构图、思维导图、ER图、时序图）。生成 XML → HTML 嵌入 → Python 本地服务 → 浏览器自动渲染预览。支持对话式实时编辑。

## Key Concepts
- **Diagram Generation** — 自然语言自动生成专业图表
- **draw.io** — 开源图表格式和渲染器
- **Natural Language to Diagram** — 文本描述到可视化图表的转换
- **Claude Code Plugin** — Claude Code 生态工具
