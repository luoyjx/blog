## Summary
26.3k stars 的 Next.js Web 应用，自然语言+AI 生成 draw.io 图表。支持图片/PDF 上传提取内容、AI 推理展示（o1/o3/Gemini/Claude）、图表版本管理、AWS/GCP/Azure 云架构图、动画连接器。集成14+ AI 提供商，提供 **MCP Server** 供 Claude Desktop/Cursor/VS Code 使用。技术栈：Next.js 16 + React 19 + Vercel AI SDK。

## Key Concepts
- **AI Diagram Generation** — LLM 驱动的自然语言到图表转换
- **draw.io** — 开源图表格式，XML 渲染
- **Natural Language Visualization** — 文本描述到可视化的 AI 桥接
- **MCP Server** — 作为 MCP 服务集成到 AI 编码工具
- **Multi Provider AI** — 14+ AI 提供商支持
