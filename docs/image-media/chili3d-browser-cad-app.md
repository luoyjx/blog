## Summary
4.4k stars 的开源浏览器 3D CAD 应用，TypeScript 构建，OpenCascade 编译为 WebAssembly + Three.js 渲染。支持布尔运算、草图、拉伸、扫掠等完整 CAD 工作流，STEP/IGES/BREP 格式互导，中英文界面，AGPL-3.0 开源。

## Key Concepts
- **Chili3D** — 浏览器端 3D CAD，无需安装（4.4k stars）
- **WebAssembly CAD** — OpenCascade 编译为 WASM 在浏览器运行
- **3D Modeling** — 草图、拉伸、布尔运算、扫掠、放样

## Capabilities
| 类别 | 功能 |
|------|------|
| 建模 | 基础形体、2D草图、布尔运算、拉伸/旋转/扫掠/放样 |
| 编辑 | 倒角、圆角、裁剪、变换 |
| 精度 | 捕捉、轴追踪 |
| 格式 | STEP / IGES / BREP |
