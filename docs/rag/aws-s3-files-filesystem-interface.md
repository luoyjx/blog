## Summary
AWS 新功能，为 AI Agent 提供对 S3 资产的文件系统接口。Agent 可用标准文件系统语义（读/写/列目录）操作 S3 对象，降低 Agent 访问云存储的复杂度。

## Key Concepts
- **AWS S3 Files** — S3 资产的文件系统语义接口
- **Agent Storage** — AI Agent 状态持久化和共享内存
- **Filesystem Interface** — 标准化 Agent 与存储层的交互
