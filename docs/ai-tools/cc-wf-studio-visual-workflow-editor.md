## Summary
VS Code 扩展，提供拖拽式 **Visual Workflow Editor** 设计 AI Agent 工作流。支持多 Agent 编排、子 Agent、Skills 和 **MCP Integration**。核心亮点：可用自然语言对话迭代改进工作流（内置 MCP Server 自动启动），一键导出为 Claude Code、Copilot、Cursor、Gemini CLI 等框架可用的格式。基于 **React Flow** 构建，工作流存储为 JSON（`.vscode/workflows/`）。4.7k stars，AGPL-3.0。

## Key Concepts
- **Visual Workflow Editor** — 拖拽画布设计 Agent 编排，无需代码
- **Agent Orchestration** — 多 Agent、子 Agent、Skills 的可视化组合
- **MCP Integration** — 内置 MCP Server 支持 AI 辅助编辑工作流
- **React Flow** — 底层画布渲染库
- **No Code Agent Design** — 降低 Agent 工作流设计门槛
