## Summary
Vercel Labs 的 Rust 原生 **Browser Automation** CLI，无 Node.js/Playwright 依赖。核心特色：语义定位（**Accessibility Tree** + @e1 引用）适合 AI 处理、**Network Interception**（mock/拦截/HAR录制）、多标签页、批量命令、AI Chat 自然语言控制模式。安装支持 npm/Homebrew/Cargo。

## Key Concepts
- **Browser Automation** — Rust 原生高性能浏览器控制
- **Rust CLI** — 无 JS 依赖的原生 CLI
- **Accessibility Tree** — 语义元素定位，AI 友好
- **AI Agent Browser** — AI Agent 专用浏览器自动化
- **Network Interception** — 请求拦截、mock、HAR 录制
