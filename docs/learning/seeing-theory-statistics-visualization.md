## Summary

Brown University出品的开源统计学可视化网站 Seeing Theory，用D3.js制作交互式动态图，覆盖基础概率到贝叶斯推断的完整概率统计知识体系。有中文版本，提供配套PDF教材。适合入门和复习，解决了公式驱动学习导致"满页公式把人绕到失去方向"的问题。

## Key Concepts

- **统计可视化** — 用D3.js交互图让统计概念可见，降低理解门槛
- **概率论** — 从随机事件、期望、方差到条件概率的完整基础覆盖
- **贝叶斯推断** — 贝叶斯公式、似然、先验与后验的可视化演示
- **中心极限定理** — 概率分布部分的核心内容之一
