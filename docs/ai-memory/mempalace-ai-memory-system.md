## Summary
AI 记忆系统，灵感来自古希腊记忆宫殿（Method of Loci），SOTA 基准表现，MIT 开源。女演员 Milla Jovovich 参与。核心思路：将语义内容组织为空间化层级结构，通过空间导航检索。

## Key Concepts
- **Memory Palace** — 古希腊记忆术，将信息与空间位置关联
- **Method of Loci** — 记忆宫殿的具体技法
- **Spatial Memory** — 空间化组织的 AI 记忆结构
- **AI Memory System** — 应用记忆宫殿原理的 LLM 记忆系统

## Method of Loci → AI 映射
| 传统 | AI 版本 |
|------|---------|
| 空间位置 | 语义节点 |
| 空间导航 | 层级检索 |
| 位置关联 | 语义关联 |
