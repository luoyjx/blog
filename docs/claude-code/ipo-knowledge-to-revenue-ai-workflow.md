## Summary
IPO 模型（Input-Processing-Output）是一套利用 AI 工具实现知识变现的系统化工作流。输入阶段将多元信息源转为 Markdown，处理阶段用 NotebookLM 和 Claude Code 生成问答内容，输出阶段通过思维导图和演示创作原创内容。核心理念是"用 AI 做研究，用人做表达"，AI 负责信息处理，人类主导最终的创意表达。整个流程每日耗时约 3-5 小时。

## Key Concepts
- **IPO Model** - Input-Processing-Output 知识变现框架
- **Knowledge Monetization** - 知识转化为收入的系统方法
- **AI Assisted Research** - AI 辅助的研究与信息处理
- **Content Creation Pipeline** - 内容创作的流水线工作流

## Detailed Content

### Input 输入阶段（0.5-1 小时）

将多元信息源转换为 Markdown 文档：
- YouTube to **NotebookLM**
- **Obsidian** Web Clipper
- Cloud Document Converter

### Processing 处理阶段（1-2 小时）

使用 **NotebookLM** 知识库 + **Claude Code** 生成全面的问答内容，完成信息的深度加工和结构化。

### Output 输出阶段（1-2 小时）

通过多种形式创作原创内容：
- 思维导图
- 语音转录
- Keynote 演示

### 核心理念

> "用 AI 做研究，用人做表达"

提倡人类主导表达，AI 负责研究和信息处理。不直接使用 AI 生成的幻灯片和脚本，而是让人类完成最终的创意表达和呈现。
