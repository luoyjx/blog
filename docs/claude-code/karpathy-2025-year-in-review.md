## Summary
Karpathy 2025年度六大主题：**RLVR**（新训练阶段，自动奖励验证发展推理策略）、**Jagged Intelligence**（LLM 智能不均匀）、**LLM Application Layer**（Cursor 代表的新软件层：上下文工程+多LLM编排）、**Claude Code**（本地运行的 Agent 式工具）、**Vibe Coding**（自然语言编程民主化）、**Nano Banana**（多模态GUI范式）。结论：行业仅挖掘了10%潜力，"the field feels wide open"。

## Key Concepts
- **RLVR** — Reinforcement Learning from Verifiable Rewards
- **Jagged Intelligence** — LLM 能力分布不均匀现象
- **LLM Application Layer** — 上下文工程+编排+界面+自主控制的新软件层
- **Vibe Coding** — 非专业人员用自然语言创建程序
- **Claude Code** — 本地运行的 Agent 式编码工具
- **Nano Banana** — 多模态文本+图像 GUI 范式
