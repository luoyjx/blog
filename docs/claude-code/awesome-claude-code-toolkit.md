## Summary
Awesome Claude Code Toolkit 是 Claude Code 最全面的工具集合，包含 135 个 Agent、35 个精选 Skill（加上 40 万社区 Skill）、42 个命令、176+ 插件、20 个 Hook、15 个规则、7 个模板、13 个 MCP 配置和 26 个配套应用。项目涵盖了 Claude Code 生态系统的各个方面，从专业角色 Agent 到生命周期自动化 Hook，是发现和选择 Claude Code 扩展的一站式资源。

## Key Concepts
- **Claude Code Ecosystem** - Claude Code 的完整工具生态系统
- **Plugins** - 176+ 领域专用扩展
- **Agents** - 135 个，分为 10 类专业角色
- **Skills** - 35 精选 + 40 万 SkillKit 社区贡献
- **Hooks** - 20 个生命周期自动化钩子
- **MCP Config** - 13 个外部工具集成配置
