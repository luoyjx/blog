## Summary
Rust 实现的进程内 SQL 数据库，18.1k stars。完整 SQLite 兼容（方言/文件格式/C API）+ 现代增强：**Vector Search**、CDC、全文搜索（Tantivy）、**io_uring** 异步 I/O、加密存储、内置 **MCP Server** 模式。多语言绑定（Go/JS/Java/.NET/Python/Rust/WASM）。适合嵌入式/边缘/分布式场景。

## Key Concepts
- **Turso** — SQLite 兼容的现代嵌入式数据库
- **SQLite Compatible** — 完整方言和文件格式兼容
- **Vector Search** — 内置向量精确搜索
- **MCP Server** — AI Agent 可通过 MCP 直接操作数据库
- **Embedded Database** — 进程内运行，无需独立服务器
