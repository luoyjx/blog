## Summary
OpenAI 与哈佛经济学家联合发布的最大规模 ChatGPT 使用研究（NBER #34255），分析110万对话（2024.5—2025.7）。核心发现：70%用途非工作；最大单类是"实用指导"(29%)；用户最看重"顾问"角色(49%询问类)；AI采用正在快速民主化——低收入国家增速4倍于高收入国家。

## Key Concepts
- **ChatGPT Usage** — 110万对话的使用模式研究
- **AI Adoption** — 28%在职者工作中使用，两年前仅8%
- **NBER Study** — 学术级别用户行为研究

## Usage Breakdown
**三大类别**（占75-80%）：
| 类别 | 占比 |
|------|------|
| 实用指导（辅导/建议/创意） | 29% |
| 写作（2/3是修改，非创作） | 24% |
| 信息获取（增长最快） | 24% |

**交互模式**：
| 模式 | 占比 |
|------|------|
| Asking 询问（用户评分最高） | 49% |
| Doing 执行 | 40% |
| Expressing 表达 | 11% |

## Key Findings
- 70% 非工作用途，30% 工作相关
- 低收入国家增速是高收入国家 **4倍**
- 女性用户从37%增至52%（2024→2025）
- 编程和自我表达仍是小众用途
